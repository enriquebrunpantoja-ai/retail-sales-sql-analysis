# Retail Sales SQL Analysis

## Project Overview

This project analyzes retail sales data using SQL to answer key business questions related to revenue, customer behavior, and product performance.

## Dataset

The dataset consists of three tables:

- customers
- products
- orders

## Business Questions

- Which product generates the most revenue?
- Which customers spend the most?
- Which customers spent more than 1000?
- How are customers segmented by value?
- Which country generates the most revenue?

## SQL Skills Demonstrated

- JOIN
- GROUP BY
- HAVING
- CASE WHEN

## Key Findings

- Phone is the top revenue-generating product.
- Customer 1 is the highest spender.
- Revenue is concentrated in a small number of customers.
- Most customers are Low Value.
- USA generates the highest revenue.
